# Idea Manager

Built with Next.js and PostgreSQL to track ideas in an efficient and simplistic manner.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
